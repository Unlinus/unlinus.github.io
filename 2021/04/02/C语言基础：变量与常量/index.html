C语言基础：变量与常量

发表于2021-04-02|更新于2021-04-07|C语言
|阅读量:|评论数:

定义

变量和常量是程序处理的两种基本数据对象。

变量

变量简介

变量即程序可操作的存储空间。程序可以通过直接调用变量名，来使用变量里面的数据。

变量命名方式

变量的命名方式有明确的要求。

  1. 变量名只能由26个大/小写字母、数字、下划线组合而成。
  2. 变量名的首字符必须是字母或下划线。例如test123、_killer；
  3. 变量名区分大小写。例如，Unlinus和unlinus是两个不同的变量。
  4. 不能使用关键字给变量命名。

关键字

截至目前为止，C语言的关键字共有44个。

  • ANSI C：
    auto break case char const continue default do
    double else enum extern float for goto if
    int long register return short signed sizeof static
    struct switch typeof union unsigned void volatile while
  • C99新增关键字（1999年）：
    inline restrict _Bool _Complex _Imaginary
  • C11新增关键字（2011年）：
    _Alignas _Alignof _Atomic _Static_assert _Noreturn Thread_local _Generic

数据类型

前面讲过，变量是程序可操作的存储空间。那么，这个存储空间的大小我们称之为数据类型。数据类型用于存放不同类型的数据，以提高存储空间的利用率。常见数据类型有:

  1. 整型(int)，反映了所在主机中整数的最自然长度（主机不同，长度也不同）；
  2. 字符型(char) ，只占用一个字节；
  3. 单精度浮点型(float) ；
  4. 双精度浮点型(double) 。
